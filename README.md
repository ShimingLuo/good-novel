# 📖 Good — AI 小说创作 IDE (Web 版)

AI 驱动的小说创作集成开发环境，纯 Web 端工具，使用本地文件存储。

## ✨ 核心功能

### 🧬 AI 全流程创作管线
- **一键配置生成** — 从一句话灵感生成完整小说配置
- **四步架构生成** — 故事前提 → 角色图谱 → 世界观 → 情节大纲
- **章节蓝图** — AI 自动生成每章的详细执行细纲
- **流式正文生成** — 基于蓝图和上下文生成章节正文
- **AI 精修** — 将草稿提升到大神级质量
- **AI 审稿** — 一致性检查、剧情逻辑审查
- **定稿后处理** — 自动生成章节要点 + 更新角色卡状态

### 🤖 Agent 系统
- **ReAct 循环** — 推理 + 行动的多轮工具调用
- **项目感知** — 自动读取架构、角色卡、章节等项目数据
- **Skills 模块** — 可扩展的知识技能包（内置 5 个 + 自定义）

### 👤 角色卡管理
- 完整的角色档案（外貌、性格、背景、能力等）
- **动态状态追踪** — 位置、能力等级、身体/心理状态、关键道具
- 定稿后自动更新角色状态

### 🔌 模型兼容
- OpenAI / DeepSeek / Gemini / Ollama / 智谱 GLM
- 任何 OpenAI 兼容 API（BYOK）

## 🚀 快速开始

```bash
# 安装依赖
npm run install:all

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173

## ⚙️ 配置

1. 启动后点击右侧 Agent 面板的 ⚙️ 设置
2. 填入 API Key 和 Base URL
3. 选择模型，保存即可

## 📋 创作阶段工作流

1. **创建项目** → 填写基本信息或使用 AI 一键生成
2. **生成架构** → 四步流水线自动生成故事骨架
3. **生成蓝图** → AI 为每章生成详细执行细纲
4. **写作正文** → 基于蓝图流式生成章节
5. **精修** → AI 精修提升文笔质量
6. **审阅** → AI 一致性审查
7. **定稿** → 自动生成章节要点 + 更新角色卡状态
8. **Agent 辅助** → 随时用 Agent 进行分析和创意辅助

## 🧩 内置 Skills

| Skill | 功能 |
|---|---|
| 章节审阅 | 剧情逻辑、角色一致性、节奏感、伏笔呼应、文笔风格 5 维度评分 |
| 脑暴创意 | 围绕话题生成 5+ 个创意方向 |
| 角色分析 | MBTI/大五人格、深层动机、角色弧、关系网络、冲突点 |
| 连续性检查 | 时间线、地理、角色状态、设定遵守、伏笔追踪 |
| 写作教练 | 叙述技巧、描写技法、对话写作、节奏控制、悬念设置 |
